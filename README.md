# ✨ Tweaks — Immersive Social Feed

![Version](https://img.shields.io/badge/version-1.0.0-purple)
![License](https://img.shields.io/badge/license-MIT-blue)

A full-screen, vertical swipeable social media web app inspired by TikTok and Instagram Reels. Built with vanilla HTML/CSS/JS and Tailwind CSS.

## 🚀 Features

| Feature | Description |
|---------|-------------|
| **Vertical Swipe Feed** | Snap-scrolling video/image posts |
| **Twimojis** ✨ | Click emojis → they float across the screen! |
| **24h Stories** | Stories with animated gradient rings |
| **Live Chat Panel** | Slide-out chat with auto-replies |
| **Like Animation** | Heart burst effect + haptic feedback |
| **Dark/Light Mode** | Toggle themes |
| **Hold to Upload** | Long-press FAB with progress ring |
| **Haptic Feedback** | Vibrations on supported devices |
